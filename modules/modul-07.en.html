Claude in Microsoft Office (Excel, PowerPoint & Word)

The Claude Add-in in Excel, PowerPoint and Word: generate formulas by voice, write VBA macros, fix circular references, create presentations from data tables and review contracts with tracked changes.

30 min. learning time 11 quiz questions Includes video

1. Two Operating Modes

ModeAccessKey advantage
Standalone chatBrowser or desktop appFlexible all-purpose solution — paste data manually
Office Add-inDirectly in Excel / PowerPoint / WordClaude sees the open file and can directly read and edit cells, slides or text passages
The decisive difference

The Add-in reads the open document context automatically — Claude knows which cells are selected and can intervene directly, without any copy-pasting.

2. Setup

  • Search for "Claude" in the Microsoft AppSource Store and install with one click.
  • Requires an active Claude Pro or Team subscription. The new Word Add-in (see below) currently requires a Team or Enterprise contract.
  • No IT installation required — standard Office Add-in, works immediately.

3. Claude in Excel

Generate formulas by voice

Describe what you need in natural language — Claude writes the exact formula and places it in the target cell. No formula syntax knowledge required.

Transform and clean data

Remove duplicates, split columns, fill missing values, standardise date formats — all by natural language instruction, applied directly to the sheet.

Write VBA macros

Claude writes complete, working VBA code. Important limitation: Claude cannot execute the macro — the user starts it manually via the VBA editor or a button.

Detect and fix errors

Claude immediately identifies circular references, explains the cause in plain language and corrects the cell directly.

Accept all edits mode

When activated, Claude applies all corrections directly to the sheet without asking for confirmation after each change — ideal for bulk transformations.

4. Claude in PowerPoint

  • Generate complete slides from a data table or written outline.
  • Automatically add speaker notes to every slide.
  • Shorten, rephrase or bring text into corporate design language.
  • Suggest and create appropriate chart types from data tables.

5. New: Claude in Word (Beta)

Since April 2026, Claude for Word has completed the "classic Office trio" alongside Excel and PowerPoint:

  • Document review: Check spelling, style and consistency even in long contracts or reports.
  • Redlining: Suggest and comment on contract clauses, marking them as suggested changes.
  • Tracked changes: Claude works directly with Word's own change-tracking — every edit stays traceable.
Requirement

Claude for Word is in beta and currently only available with a Team or Enterprise contract — not yet on Pro.

6. Shared Context Across the Add-ins

Since March 2026, Excel and PowerPoint share a common context: Claude can take figures from an open Excel sheet and use them directly for a PowerPoint slide — no copy-paste, no re-explaining. Word's integration into this shared context is expected to follow.

The same update also introduced Skills directly inside Excel and PowerPoint — reusable workflows that teams can save and share as one-click actions (e.g. "create monthly report").

7. Data Security

The Add-in transmits only the selected cell range or text passage to Anthropic's servers — not the entire workbook or document. Background worksheets and non-selected ranges are not transmitted.

Data protection at IEG

No real client data or deal terms in the Add-in. Anonymise sensitive models first, or use the IEG Claude Enterprise Account for financial models with real figures.

8. Integration Limits

  • No direct macro execution — manual start via VBA editor required.
  • No database access (SAP, Bloomberg, SQL) — data must be exported first.
  • Only visible context — not all background worksheets are available.
  • No real-time synchronisation with live data sources.
  • Word Add-in still in beta — feature set is expanding continuously, currently Team/Enterprise only.
Practical exercise

Apply what you've learned

Use the Claude Add-in to solve real Excel and PowerPoint tasks.

Exercise 1

Formula generation and data cleaning in Excel

Use the Add-in to solve tasks that would normally require formula knowledge.

  • Open a spreadsheet with a data table (at least 20 rows).
  • Select a column and ask Claude to calculate the sum, average and count in natural language.
  • Ask Claude to remove all duplicate rows.
  • Ask Claude to add a new column that classifies each row as "High", "Medium" or "Low" based on a value threshold you define.
  • Enable "Accept all edits" and ask Claude to standardise all dates to DD/MM/YYYY format.
Goal: Complete five tasks that would normally require formula knowledge — all in natural language, directly in the sheet.
Exercise 2

Slide generation in PowerPoint

Turn a data table into a presentation slide.

  • Open PowerPoint and install the Claude Add-in if not yet done.
  • Create a blank slide.
  • Ask Claude to create a summary slide from a table you paste into the chat (3–5 rows of data).
  • Ask Claude to add speaker notes that explain the key insight from the data.
  • Ask Claude to suggest an appropriate chart type for the data.
Example prompt:

Create a slide summarising this data: [paste your table]. The slide should have a clear headline, a bullet-point summary of the 3 key findings, and speaker notes explaining what the audience should take away.

Goal: From raw data to presentation-ready slide in under 2 minutes.
