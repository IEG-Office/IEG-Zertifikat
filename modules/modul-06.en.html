Claude Cowork — The Autonomous Desktop Agent

While a chatbot only responds, Claude Cowork acts independently on your computer: sorting files, creating documents, researching online and planning tasks — including setup, security and personal instructions.

45 min. learning time 10 quiz questions Includes video

1. What is Claude Cowork?

Claude Cowork is an AI agent that acts independently on your computer. The model is the brain, Cowork is the hands — it opens, moves, renames and controls files and programmes. Unlike the web chat, Cowork has direct access to your local environment.

2. Setup and Security

  • Desktop app: Requires the Claude desktop app (Mac/Windows) and a Pro subscription.
  • Security sandboxing: Cowork only works in folders you explicitly authorise — all other data on your computer remains untouched.
  • Working environment: Create a dedicated project folder for each task to keep Cowork's work organised.
Important

Only authorise folders that contain relevant project files. Never authorise your entire hard drive or sensitive system folders.

3. Core Functions

  • File management: Analyse, rename and sort hundreds of files according to defined rules.
  • Document creation: Create Excel tables, CSV trackers or PowerPoint presentations from raw data.
  • Web browsing: Via "Claude in Chrome" — research online, summarise pages, extract structured data from dashboards.
  • Connectors: Link to Gmail, Notion, Google Drive or Slack — but only via the IEG Claude Account.

4. Advanced Automation

  • Scheduling: e.g. "Check the minutes folder every morning at 8am and send any new files to the team."
  • Markdown memory: Cowork has no built-in session memory and stores information in Markdown files in the project folder — these persist across sessions.
  • Skills & Plugins: Teach Cowork specific domain expertise or store your corporate design for consistent outputs.

5. Important Rules

Session persistence

The desktop app must remain open while a task is running. Closing the app stops the agent — the task does not continue in the cloud.

Model choice

Sonnet for simple sorting and organisation tasks. Opus for complex analyses that require deeper reasoning — note that Opus uses more credits.

Practical exercise

Apply what you've learned

Set up Cowork for a real task and run your first autonomous workflow.

Exercise 1

Set up your first Cowork project

Configure Cowork safely before running any tasks.

  • Create a dedicated folder on your desktop: "Cowork_Test".
  • Copy 5–10 sample files into it (documents, PDFs, or spreadsheets — no sensitive data).
  • Open the Claude desktop app and navigate to Cowork.
  • Authorise only the "Cowork_Test" folder — nothing else.
  • Ask Cowork to list all files and organise them by file type into subfolders.
Goal: Experience the sandboxing principle firsthand — Cowork only touches what you allow it to touch.
Exercise 2

Run a document creation task

Give Cowork raw data and let it produce a structured output.

  • Place a raw data file (e.g. a CSV or unformatted Excel export) in your Cowork folder.
  • Instruct Cowork to clean the data and create a formatted summary table.
  • Ask it to add a brief written summary of the key figures at the top.
  • Check the output: is it accurate? What would you adjust?
Example prompt:

In my Cowork_Test folder there is a file called "raw_data.csv". Please clean it, remove duplicates, and create a formatted Excel summary with the top 10 entries by value. Add a 3-sentence executive summary at the top of the sheet.

Goal: From raw data to polished output — without opening Excel manually.
