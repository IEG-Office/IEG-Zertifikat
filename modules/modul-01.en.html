Claude — The Complete Beginner's Guide (2026)

Chat discipline, web search, file analysis, technical support, personalisation and data privacy — a solid foundation for every professional start with Claude. (You have already been introduced to the model family in Module 0.)

45 min. learning time 10 quiz questions Includes video

1. Fundamentals and Chat Discipline

Claude is an AI tool from Anthropic, operated through a simple text box. A key success factor is chat discipline: start a new chat for every new topic. Mixing topics in a single chat confuses the AI and significantly degrades the quality of results.

The reason lies in what is known as the context window: with every response, Claude re-reads the entire chat history from the beginning. The longer and more topically unfocused a chat becomes, the more irrelevant information the model has to process — answers become less accurate and slower. A fresh chat per topic keeps the context clean and focused.

Golden rule

One topic — one chat. Mixing holiday planning with quarterly reports in the same chat produces worse results for both.

Two further habits pay off from day one:

  • Provide context: Tell Claude who you are, who the output is for and what format you need — e.g. "Write as an analyst for an internal memo, maximum 200 words."
  • Work iteratively: A first response is rarely perfect. Refine in the same (topic-consistent) chat with follow-up instructions such as "shorter", "more formal" or "with references".

2. Core Working Functions

The most important tools can be activated via the plus icon below the text input. These three form the backbone of daily work:

Web search

Claude can search hundreds of sources in seconds to compile current data (e.g. DAX revenues or daily industry news) into tables. Unlike pure model knowledge, results are backed by source links — always verify these for critical figures. Formulate requests precisely with a time period and desired output format, e.g. "Summarise Q3 revenues of the three largest German car manufacturers as a table with source references."

File analysis

PDFs, Excel or Word documents can be uploaded directly. Pro tip: use the stopwatch icon (extended thinking mode) so Claude takes more time to read the file thoroughly and provide more precise answers. Ask specific questions about the document rather than a blanket summary — e.g. "What three risks does the report mention on pages 10–15?" With multiple files, Claude can also compare them (e.g. contrast two quarterly reports).

Technical support

Claude helps with PC and software problems. The most efficient workflow: first describe the problem in general terms, then upload a screenshot of the user interface (e.g. the activity monitor or an error message) — Claude reads the screen contents and diagnoses the cause directly. Then ask for a step-by-step guide and have technical terms explained in plain language if needed.

3. Personalisation: Tailoring the AI to You

To make Claude sound less generic and more suited to your working style, there are three tools:

  • Preferences: Global rules in the settings (e.g. "Always respond briefly and in English" or "I am a Mac user"), automatically applied to every new chat.
  • Styles: You can choose predefined styles (formal, concise) or create a custom style by uploading your own writing samples (e.g. previous memos or scripts) — Claude analyses tone, structure and word choice and writes in your style from then on.
  • Memories: Using the command "Please create a memory", Claude permanently stores facts about you (e.g. "I have a fear of flying" or "My team works in M&A"), which it automatically takes into account in future chats.

4. Pro Features: Programming and Excel

Programming without code

Claude can create interactive dashboards or complete HTML websites — including contact forms — without any programming knowledge. On request, Claude even provides a step-by-step guide for publishing the page.

Claude in Excel

The Excel Add-in (Pro feature) allows Claude to work directly in spreadsheets: retrieve data from the web, create dynamic charts linked to cells and independently correct complex errors such as circular references — all via natural language instructions.

5. Security and Data Privacy

  • Disable training: Go to settings under "Privacy" and switch off "helps improve Claude" so your inputs are not used to train new models.
  • Incognito mode (ghost icon): Chats are not permanently stored in the sidebar — ideal for private or one-off requests.
Mandatory at IEG

Disabling the training option is mandatory at IEG. Do it before your first use — once set, it remains permanently active.

Exercise 1

The right function for the right task

Web search, file analysis or screenshot upload? Practise choosing the right Claude function for different work situations and combining it with a precise instruction.

  • Create a table with three columns: Task, Function and Rationale.
  • Evaluate these scenarios: (a) Research current industry multiples with source references, (b) Analyse an 80-page market study as a PDF, (c) Diagnose a slow PC, (d) Compare two quarterly reports, (e) Draft a short internal email.
  • For one scenario of your choice, formulate the complete instruction — including context, format and source requirements.
Example prompt:

I will give you 5 tasks from everyday work. Recommend the appropriate Claude function (web search, file upload + extended thinking mode, screenshot upload, standard chat) for each and justify the choice in one sentence. Format: table with three columns.

Goal: Think function and precise instruction together — the right tool alone is not enough, the instruction must be clear.

Exercise 2

Account setup: privacy and first preference

Before working productively with Claude, your account should be correctly configured. This exercise walks through the two most important settings and tests the effect of a global preference.

  • Privacy: Open settings → Privacy → disable "helps improve Claude". Take a screenshot as proof.
  • Preference: Set a global preference useful for your daily work (e.g. "Always respond in English, structure answers with bullet points, maximum 5 sentences").
  • Then open a new chat and ask any question — check whether Claude applies the preference without being prompted.
  • Also test incognito mode: start a chat with the ghost icon and check whether it appears in the sidebar after closing.
Example prompt (for testing the preference):

Briefly explain what a DCF model is.

Goal: Ensure privacy and configure Claude so it responds exactly as you need from now on — without any briefing.
Approach
  • Privacy: Settings (gear icon) → Privacy → uncheck "helps improve Claude" → Save.
  • Preference example: "Always respond in English. Structure answers as a bullet list. Keep it to a maximum of 5 points." → Active from the next chat onwards.
  • Effect test: The DCF prompt should now come back in English as a bullet list — without you mentioning it in the request.
  • Incognito: Ghost icon → start chat → close window → chat does not appear in the sidebar.

Conclusion: These two settings take 2 minutes but save time every day and protect company data — done once, effective permanently.
